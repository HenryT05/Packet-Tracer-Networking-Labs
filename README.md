# Packet Tracer Networking Labs

A collection of hands-on networking labs built in Cisco Packet Tracer.
These labs are part of my journey learning networking fundamentals for
a career in cybersecurity.

## Labs

| # | Lab | Concepts Covered |
|---|-----|-----------------|
| 01 | [Basic LAN with Two PCs and a Switch](./Lab-01-Basic-LAN/README.md) | Layer 2 Switching, ARP, Static IP, Ping |
| 02 | [Routed Network](./Lab-02-Routed-Network/README.md) | Routing, Inter-VLAN, Layer 3 |
| 03 | [DHCP Setup](./Lab-03-DHCP-Setup/README.md) | DHCP, Dynamic IP, Scopes |

## Tools Used
- Cisco Packet Tracer 8.x
- GitHub for documentation
